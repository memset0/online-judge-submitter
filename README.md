# memset0 / online-judge-submitter

一个**正在开发的**为你的服务的OJ交题器



### 使用方法

1. 安装`Python 2.7`

   对于`Windows`用户可以上官网下载`msi`安装包，`Linux`标配自带

2. 安装`requests`库

   为了~~省力气~~更好的兼容性，我们将会使用`requests`库，使用

   ```shell
   pip install requests
   ```

   或

   ```shell
   pip27 install requests
   ```

   来安装。
